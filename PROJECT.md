# Проект MVR Player

MVR Player — кроссплатформенная программа на Python для просмотра файлов `.mvr`
и конвертации их в `.mp4`.

## Цели

- Создать понятную архитектуру Python-приложения.
- Подготовить основу для графического интерфейса на Tkinter.
- В будущем добавить просмотр `.mvr` файлов.
- В будущем добавить конвертацию `.mvr` в `.mp4` через FFmpeg.

## Текущий этап

На текущем этапе реализован только каркас проекта и запуск пустого окна
с названием **MVR Player**.

Логика плеера, парсинга `.mvr`, работы с FFmpeg и конвертации пока не
реализована.

## Правила разработки

- Хранить код приложения внутри `src/mvr_player`.
- Не смешивать UI, логику плеера и логику конвертации в одном модуле.
- Добавлять тесты для новой публичной логики.
- Не добавлять тяжелые зависимости без явной необходимости.
- Документировать пользовательские сценарии в `docs/USER_GUIDE.md`.
